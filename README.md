# Docker Full Stack App 🚀

A beginner-friendly full stack application containerized using Docker and Docker Compose.

## 🛠️ Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Docker
- Docker Compose

## 📦 Features

- Frontend and Backend communication
- MongoDB integration
- Dockerized full stack setup
- Multi-container architecture

## 🚀 Run Project

```bash
docker compose up --build
```

## 📸 Project Output

Frontend:
- http://localhost:5173

Backend:
- http://localhost:5000
